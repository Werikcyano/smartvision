Visão Geral do Projeto

🎯 Objetivo Principal

O Smart Vision é uma solução completa que combina Inteligência Artificial, Internet das Coisas (IoT) e Visão Computacional para criar um sistema avançado de monitoramento veicular com foco em segurança do trabalho e automação industrial.

Segurança do Trabalho

Detecção de EPIs e zonas de risco

Monitoramento Veicular

Controle inteligente de energia e GPS

Inteligência Artificial

Modelos YOLO para detecção em tempo real

dev_ceia_digefx

Pesquisa & Desenvolvimento em IA

Centro de desenvolvimento dos modelos de visão computacional e scripts de processamento avançado.

Python YOLOv13 YOLOv6 PyTorch OpenCV
  • Modelos YOLO treinados para detecção de EPIs
  • Scripts de processamento em lote de vídeos
  • Análise de zonas de segurança baseada em centróide
  • Sistema de tracking para violações de segurança
  • Estimativa 3D de distâncias e proximidades
Classes Detectadas:
PESSOA, COM_CAPACETE, COM_LUVA, CONE, SINALIZACAO, FUMANDO_CIGARRO, SEM_VIDEO, ESCADA

digefx-esp

Firmware ESP32 para IoT Veicular

Sistema embarcado para controle inteligente de energia e telemetria veicular com GPS integrado.

C++ Arduino PlatformIO ESP32 GPS
  • Controle de 2 relés baseado em ignição/voltagem
  • Monitoramento contínuo de bateria com proteção
  • Sistema GPS com otimização para fixação rápida
  • 6 LEDs de status + buzzer para feedback
  • Comunicação serial configurável via comandos
  • Persistência de configurações em memória não-volátil
Configurações:
Voltagem mínima, tempos de relé, ID do dispositivo via protocolo serial

digefx-monitor-api

Backend FastAPI com Sistema Event-Driven

API robusta com arquitetura orientada a eventos para processamento paralelo de alertas em tempo real.

FastAPI SQLAlchemy PyTorch MQTT AMQP
  • Sistema event-driven para processamento paralelo
  • Autenticação JWT com controle de sessão
  • Monitoramento de sistema (CPU, RAM, temperatura)
  • Comunicação serial bidirecional com ESP32
  • Handlers para MQTT, AMQP, Database e Frigate
  • Processamento escalável de múltiplas câmeras
Event Handlers:
DatabaseHandler, MQTTHandler, AMQPHandler, FrigateHandler executam em paralelo

digefx-monitor-frontend

Interface Web Moderna com Next.js

Dashboard responsivo para monitoramento em tempo real com mapas GPS e controle remoto do sistema.

Next.js 15 React 19 TypeScript Material-UI Leaflet
  • Dashboard com métricas em tempo real (polling 10s)
  • Autenticação NextAuth.js integrada com backend
  • Mapas interativos com rastreamento GPS
  • Terminal web para comandos remotos
  • Gerenciamento completo de câmeras
  • Interface responsiva com Material-UI
Funcionalidades:
Status de câmeras, trajetos GPS, alertas em tempo real, controle de sistema
